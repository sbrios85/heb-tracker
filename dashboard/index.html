Sync to GitHub

Push your tracked & dismissed lists to data/tracked.json in the repo, so the daily refresh workflow can read them.

This calls a GitHub Actions workflow. You need a fine-grained Personal Access Token with Actions: read & write and Contents: read & write on sbrios85/heb-tracker. Saved locally only.

Compact data

Permanently remove dismissed products from data/details.json and data/products.json, shrinking the dashboard.

This runs the compact_dismissed GitHub Action. Dismissed-list still survives, so any product re-added by H-E-B in the future stays hidden.

Block this brand?

This runs the block_brand GitHub Action, which:

  • Adds the brand to data/brand_blocklist.json
  • Physically removes all its products from data/details.json and data/products.json
  • Skips this brand on any future discovery runs

To unblock later, edit brand_blocklist.json directly in the repo.
